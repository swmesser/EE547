Course Title: EE 447 QL Inter/Intranet App Development (Summer 2023)
Course Instructor (CI): Gregory Myers (EEC253)
Teaching Assistant(s) (TA): N/A
Course Description: This course covers development of software models and applications using Internet/Intranet technologies. Topics include: web client-server relationships, multi-tier design models, scripting and validation, basic TCP/IP networking, separation of concerns, markup and data description languages. Projects will allow the opportunity for the use of a range of tools and development platforms.

Course Learning Outcomes
1. The student will be able to apply the factual knowledge, fundamental principles, or theories learned in the course.
2. The student will be able to design web applications.
3. The student will be able to test and debug web applications to identify a problem, isolate the problem to a portion of the software that is faulty and correct the defect.
4. The student will be able to discuss the impact of the web on various portions of society including the trade-o"s related to standards compliance for both accessibility and vendor-neutral application architectures.
5. The student will be able to use the web to find and use information related to their problems. The student will be able to discuss the rate of change of technology in this application area.
6. The student is aware of issues related to standards compliance for both accessibility and vendorneutral application architectures.
7. The student will be able to use software tools in website application development.
